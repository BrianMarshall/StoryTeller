StoryTeller

This is the first page of the story.

It is defined as a <div> in the <body> of your HTML document. StoryTeller knows it is the first page to show because its id is "start". All pages must have the class "page" so that StoryTeller can identify them. All other <div>s will be treated and ordinary HTML

You can use standard HTML to format your pages. For example:

  • <b>Bold</b> = Bold
  • <i>Italic</i> = Italic
  • <u>Underline</u> = Underline

You can also use lists, tables and any other standard HTML tags.

CSS can also be used to style the pages by adding to the <style> element in the <head> of the document. Ensure that ".page { display: none;}" remains unchanged as it will avoid the used seeing the source of your pages before StoryTeller has a chance to process them.

Alternatively, an external CSS file can be included. For example <link rel="stylesheet" type="text/css" href="custom.css" /> will include the file "custom.css"

Links

The player makes choices by clicking on links. These move the player to another page, just as you clicked a link on the first page to come here.

A link is defined like so: [[link("{destination id}")]]{link text}[[endLink]]

Simply replace {destination id} with the page id to jump to and {link text} with the text you would like to appear.

For example: [[link("start")]]Go back to start page[[endLink]] produces [link("start")]Go back to start page[endLink]
